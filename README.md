# JsFloppyBird
## Floppy bird Javascript Edition

![screen shot](doc/screen_shot.png)

### How to play
* Mobile
  * Touch screen : Jump 
  * Touch Pause Icon : Pause
  
* Mouse
  * Click : Jump

* Key
  * Up arrow : Jump
  * Space : Jump
  * J : Jump
  * P : Pause
  * S : Start / Resume
  * Enter : Start / Resume
